## 2. 워크플로우별 비용 효율

### 인제스트 (전체 비용의 39.8%)

| 방식 | 건당 비용 | 시간 | 품질 |
|------|----------|------|------|
| Claude Code 직접 인제스트 | ~$1.00 | 세션 내 | 높음 (교차참조, 인덱스 갱신 포함) |
| Gemini 요약 → Claude Code 통합 | ~$0.01 + 세션비용 | 2단계 | 높음 (대용량에 최적) |
| Claude Code 인프라 구축 | $10.50 (일회성) | 1일 | — (반복 불필요) |

**핵심 발견:**
- 인프라 비용($10.50)은 일회성. 앞으로의 인제스트는 건당 ~$1.00
- Gemini 요약은 건당 $0.01 미만 — 대용량 PDF는 무조건 Gemini 선처리가 효율적
- 100건 인제스트 목표 시 인제스트만으로 ~$100 → 예산 초과 가능 → Sonnet 전환 검토

### 쿼리 (전체 비용의 6.4%)

| 방식 | 건당 비용 | 지연 | 품질 |
|------|----------|------|------|
| Gemma 4 로컬 (basic backend) | $0.00 | ~44초 | 양호 (인용 포함) |
| Gemma 4 로컬 (gemma4 backend) | $0.00 | ~90초 | 좋음 (한국어 정확) |
| Claude Code 쿼리 | ~$0.27 | 세션 내 | 최고 (복합 분석) |

**핵심 발견:**
- 일상 쿼리는 Gemma 4 로컬로 충분 (비용 $0)
- 복합 분석/다중 소스 교차만 Claude Code 사용
- 쿼리는 비용 최적화 여지가 크지 않음 (이미 $0 가능)

### 린트 (전체 비용의 4.0%)

| 방식 | 건당 비용 | 범위 | 특징 |
|------|----------|------|------|
| validate-wiki.sh | $0.00 | 규칙 기반 | 프론트매터, 링크, 인덱스 검증 |
| Codex CLI 교차 검증 | $0.17 | LLM 기반 | 규칙 + 의미 검증 |
| Claude Code 린트 | ~$0.68 | LLM 기반 | 가장 포괄적 |

**핵심 발견:**
- validate-wiki.sh로 1차 필터링 후 Codex/Claude로 2차 검증이 효율적
- Codex는 Claude 대비 75% 저렴 ($0.17 vs $0.68)하면서 품질 충분
- 주간 린트 → Codex, 월간 심층 린트 → Claude Code 배분 추천

## 3. 최적 배분 전략

### 권장 워크플로우

```
대용량 소스 → Gemini 2.5 Flash 요약 ($0.01/건)
     ↓
일상 인제스트 → Claude Code Opus ($1.00/건)
     ↓
일상 쿼리 → Gemma 4 로컬 ($0.00)
     ↓
복합 분석 → Claude Code Opus (세션 비용)
     ↓
주간 린트 → Codex CLI ($0.17/회)
     ↓
월간 린트 → Claude Code ($0.68/회)
```

### 월간 예산 시뮬레이션

| 활동 | 빈도 | 건당 비용 | 월 비용 |
|------|------|----------|--------|
| 일상 인제스트 | 10건/월 | $1.00 | $10.00 |
| 대용량 인제스트 | 2건/월 | $0.01 (Gemini) + $1.00 (통합) | $2.02 |
| 로컬 쿼리 | 30건/월 | $0.00 | $0.00 |
| Claude 쿼리/분석 | 5건/월 | $0.27 | $1.35 |
| 주간 린트 (Codex) | 4회/월 | $0.17 | $0.68 |
| 월간 린트 (Claude) | 1회/월 | $0.68 | $0.68 |
| **합계** | | | **$14.73** |

예산 활용률: 29.5% — **100건 인제스트까지 $50 이내 유지 가능**

### 비용 절감 옵션 (필요시)

1. **Claude Sonnet 전환**: 인제스트를 Opus→Sonnet으로 ($1.00 → $0.20/건)
2. **로컬 인제스트**: Gemma 4로 인제스트 테스트 (품질 트레이드오프)
3. **캐시 활용**: Claude Code 프롬프트 캐시 히트 시 input $15→$1.50 (90% 절감)
