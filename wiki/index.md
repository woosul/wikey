# 콘텐츠 인덱스

## 엔티티

(없음)
- [[nanovna-v2]] — NanoVNA V2는 50kHz~3GHz 대역을 측정하는 소형 벡터 네트워크 분석기(VNA)이다. (소스: 1개)
- [[vna]] — VNA(Vector Network Analyzer)는 RF 장비의 주파수 응답 특성을 측정하는 도구이다. (소스: 1개)
- [[v2-plus4]] — V2 Plus4는 NanoVNA V2의 확장 모델로, 주파수 측정 범위가 4.4GHz까지 확장된 소형 벡터 네트워크 분석기이다. (소스: 1개)
- [[nanovna-qt]] — NanoVNA-QT는 NanoVNA 장비를 제어하고 데이터를 시각화하는 오픈소스 소프트웨어 도구이다. (소스: 1개)
- [[dji-o3-air-unit]] — DJI O3 Air Unit은 DJI의 FPV 드론 시스템에 사용되는 고성능 디지털 영상 전송 장치이다. (소스: 1개)
- [[dji]] — DJI는 중국의 드론 및 항공 촬영 기술 전문 기업이다. (소스: 1개)
- [[vector-network-analyzer]] — RF 장치의 전기적 특성을 측정하는 데 사용되는 전자 테스트 장비. (소스: 1개)
- [[nanovna-v2-plus4]] — NanoVNA V2의 확장 모델로, 최대 4.4GHz까지 주파수 측정이 가능한 소형 벡터 네트워크 분석기. (소스: 1개)
## 개념

(없음)
- [[swr]] — SWR(Standing Wave Ratio)은 전송선로에서 진행파와 반사파의 비율을 나타내는 표준 지표이다. (소스: 1개)
- [[s-parameter]] — S-파라미터(Scattering parameter)는 고주파 회로 및 장치의 전기적 특성을 정의하는 표준 파라미터이다. (소스: 1개)
- [[sma]] — SMA(SubMiniature version A)는 고주파 애플리케이션에 널리 사용되는 나사형 동축 RF 커넥터의 표준 규격이다. (소스: 1개)
- [[mmcx]] — MMCX(Micro-Miniature Coaxial)는 소형 RF 애플리케이션에 사용되는 스냅온 방식의 동축 커넥터 표준 규격이다. (소스: 1개)
- [[user-manual]] — 특정 제품이나 시스템의 사용 방법을 설명하는 문서 유형. (소스: 1개)
- [[standing-wave-ratio]] — 전송선로에서 진행파와 반사파의 비율을 나타내는 값으로, 안테나 매칭 상태를 평가하는 데 사용되는 표준 지표. (소스: 1개)
- [[impedance]] — 교류 회로에서 전류의 흐름에 대한 저항을 나타내는 전기적 특성으로, 저항, 유도 리액턴스, 용량 리액턴스를 포함한다. (소스: 1개)
- [[smith-chart]] — 고주파 회로에서 임피던스 매칭 및 전송선 특성을 시각적으로 분석하는 데 사용되는 그래픽 도구이자 기법. (소스: 1개)
- [[sma-connector]] — 고주파 애플리케이션에 널리 사용되는 나사식 결합 방식의 소형 동축 RF 커넥터 표준. (소스: 1개)
- [[mmcx-connector]] — 소형 장치에 사용되는 스냅온 결합 방식의 마이크로 미니어처 동축 RF 커넥터 표준. (소스: 1개)
- [[open-short-load-calibration]] — 벡터 네트워크 분석기(VNA)에서 정확한 측정을 위해 수행하는 표준 캘리브레이션 절차로, Open, Short, Load 세 가지 기준 상태를 사용한다. (소스: 1개)
- [[s11]] — S-파라미터 중 반사 계수를 나타내는 표준 지표. (소스: 1개)
- [[s21]] — S-파라미터 중 전송 계수를 나타내는 표준 지표. (소스: 1개)
