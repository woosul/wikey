# Claude Code 설정

Claude Code는 wikey 프로젝트를 작업할 때 사용되는 특정 개발 환경 및 도구 사용법을 정의한다. 이 설정은 wikey의 핵심 워크플로우를 구현하는 데 필요한 체크리스트 역할을 한다. [[wikey-workflow-protocol]]의 실행을 위한 구체적인 가이드라인을 제공한다.
