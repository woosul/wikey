# Claude Code

Claude Code는 Anthropic의 LLM 모델 중 하나로, 코드 생성 및 분석에 특화되어 있습니다. [[wikey]] 프로젝트에서는 메인 인제스트, 쿼리, 린트 작업에 최고 품질의 지식 합성을 위해 활용됩니다.
