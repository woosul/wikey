# File over app

[[andrej-karpathy]]의 LLM Wiki 원문에서 강조된 원칙으로, 특정 애플리케이션에 종속되지 않고 범용적인 파일 포맷(예: 마크다운)을 사용하여 지식을 관리하는 것을 의미합니다. [[wikey]]는 이 원칙에 따라 [[obsidian]]을 브라우저로 활용하면서도 마크다운을 핵심 포맷으로 채택합니다.
