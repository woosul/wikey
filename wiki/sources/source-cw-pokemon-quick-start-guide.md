# CW POKEMON Quick Start Guide

**HAND Electronics studio**

Welcome to use the CW Pocket Wizard multi-function trainer, hope to accompany you to have a good CW learning time.

## 1. Routine operation of equipment

*   **Switch on and off:** long press the power button on the right side of the top of the device, and release it after the panel indicator lights up to complete the startup operation; In the startup state, long press the power button for more than 1.5 seconds, you can turn off the power of the device, when the shutdown beep.
*   **Enter/exit menu:** Press the power button to enter the menu or sub-menu. Press the power button left and right to select menu items. By default, you can press the F4 function key to exit the function or return to the upper-level menu. Under the main menu, you can return to the main operation interface.
*   **Parameter setting:** Press the power button to enter parameter setting, and a prompt tone is displayed. Press the power button left and right to switch different parameter options. After selecting the parameter, press the power button to confirm/exit, and a prompt tone is displayed.
*   **Automatic key Sending speed setting:** In the main operation interface, you can adjust the sending speed by pressing the power button left and right. The speed range is 10-50 yards.
*   **Volume adjustment:** You can set the volume in the menu - System parameter, the volume range is 1-63, or you can adjust the volume quickly by pressing the power button to the left or right in any interface.
*   **Use of shortcut keys:** There are four shortcut keys on the right side of the screen. In the main operation interface, press each key to enter a function interface quickly. The main shortcut area is set with four shortcut functions for exercise. The system also has 1-4 shortcut areas. You can switch between each shortcut area and the main shortcut area by long pressing the f1-F4 function key. For details about the functions of each shortcut area, see the electronic user manual.

## 2. Interface Icon Description

*   **ICONS in the upper status bar:** battery power icon, startup time, decoding board connection icon, decoding filter bandwidth icon, DNR digital noise suppressor icon, current coding speed, current decoding speed, key interface type, audio icon from left to right; Decoder board ICONS are not displayed when the decoder board is off or when the host is mini.
*   **Function prompt icon for shortcut keys:** The four multi-function shortcut keys on the right correspond to the function prompt icon. The prompt icon displays the corresponding function of the key on the current page based on different interfaces and functions. Due to the size of the icon, the function prompt is a simple English word.

## 3. Q&A

*   **Q, plug in the hand button, press the ring has been didi?** A. Please check the setting of the key type. The factory default is automatic key mode, and please set the manual key mode when connecting the manual key.
*   **Q, insert auto key, dot is reverse?** A. Please set dot swap parameters in key parameter Settings.
*   **Q. Why is the decoding accuracy of professional version not high?** A. Correct CW side tone frequency and appropriate input volume. Ensure that the radio side tone frequency is set to 600Hz and accurately adjusted to CW signal frequency; Properly adjust the volume of the radio or the volume of the external MIC option, so that when there is no signal, the panel indicator just goes out, and the audio decoding sensitivity is the highest.
*   **Q, which USB port is the charging port?** A. The USB port on the professional version is the charging port.
